## Purpose

This folder contains the complete set of information security policies and procedures for CloudVault FHX, organized by NIST SP 800-53 Rev 5 control family. These documents satisfy the policy and procedure requirements across all 20 control families in the FedRAMP High baseline and are referenced throughout the System Security Plan.

## FedRAMP Requirement

FedRAMP High requires a formal policy and procedure document for each of the 20 NIST SP 800-53 Rev 5 control families. Policies must be reviewed and approved at least annually and whenever significant changes occur. Review dates and approver signatures must be visible on each document.
