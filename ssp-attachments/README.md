# ssp-attachments/ — SSP Supporting Exhibits

This folder contains the supporting exhibits and attachments that are formally referenced within the System Security Plan (SSP) for the CloudVault Federal Health Exchange (FHX). In a real FedRAMP engagement, these attachments are submitted as part of the authorization package and must be consistent with every claim made in the SSP body.

## Why Attachments Matter

The SSP attachments are not optional or decorative — they are formal components of the authorization package. If the main SSP body states "see Appendix D for interconnection details" and Appendix D is missing or inconsistent, the 3PAO will flag a documentation deficiency. In worst cases, missing attachments can delay authorization by weeks while they are corrected and resubmitted to the FedRAMP PMO.

Keeping each attachment as a separate, version-controlled file also enables efficient updates. When a new interconnection is added to CloudVault FHX, only Appendix D needs to be revised — not the entire SSP.
