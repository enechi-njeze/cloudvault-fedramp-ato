## Purpose

This folder contains the Control Implementation Summary (CIS) for CloudVault FHX. The CIS provides a concise mapping of all 421 FedRAMP High baseline controls to their implementation status, responsible party (CloudVault FHX vs. AWS GovCloud), and implementation approach. It is a required deliverable for FedRAMP package submission and is one of the primary documents reviewed by the JAB during P-ATO evaluation.

## Required Artifacts

Upload the following files to this folder:

| Artifact | Description | Format |
|---|---|---|
| Control Implementation Summary (CIS) | Full 421-control table showing responsibility, status, and implementation description | Excel or Word |
| Inherited Controls List | List of controls fully inherited from AWS GovCloud FedRAMP High package | Excel or PDF |
| Shared Responsibility Matrix | Matrix showing which controls are fully CloudVault, fully AWS, or shared | Excel or PDF |
| Customer Responsibility Matrix (CRM) | Obligations that agency customers must implement on top of CloudVault FHX | Word or PDF |

## FedRAMP Requirement

The CIS is a mandatory FedRAMP SSP appendix. It allows the AO and JAB to quickly understand the control responsibility breakdown without reading through the full SSP narrative. The CIS must be consistent with the SSP control descriptions and the SAR findings. Any discrepancy between the CIS and SSP is a POA&M item.
