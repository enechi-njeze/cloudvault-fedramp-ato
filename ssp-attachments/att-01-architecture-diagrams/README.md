## Purpose

This folder contains the official architecture diagram artifacts for the CloudVault Federal Health Exchange (FHX) System Security Plan (SSP). Upload your architecture diagram files here for reviewer access.

## Files to Upload Here

Upload one or more of the following file types:

- Architecture boundary diagram (Visio .vsdx, PDF, or PNG)
- Network topology diagram (Visio .vsdx, PDF, or PNG)
- Data flow diagram(s) (Visio .vsdx, PDF, or PNG)
- AWS GovCloud VPC layout diagram (Visio .vsdx, PDF, or PNG)
- Disaster recovery architecture diagram (Visio .vsdx, PDF, or PNG)

## Required Diagram Contents (per FedRAMP Guidance)

- Authorization boundary clearly marked, showing what is in scope vs. out of scope
- All external interconnections entering or leaving the boundary
- Network segmentation (VPCs, subnets, security groups)
- All major system components labeled with their function
- Data flow paths showing how federal data (PHI, PII, CUI, FTI) traverses the system
- Trust boundaries between tiers
- Version number and date on every diagram
