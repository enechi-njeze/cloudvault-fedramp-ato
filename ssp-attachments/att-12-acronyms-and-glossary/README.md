## Purpose

This folder contains the Acronyms and Glossary document for CloudVault FHX. This attachment ensures that all terms, acronyms, and abbreviations used throughout the SSP and authorization package are defined consistently. It is a required FedRAMP SSP appendix and serves as the reference standard for all document authors on the CloudVault FHX program.

Upload the following files to this folder:
