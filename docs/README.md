## About the System Security Plan

The SSP is the most important document in any FedRAMP authorization package. It is a living document — typically 300–500 pages in a real federal engagement — that describes every security control, how it is implemented, who is responsible for it, and what evidence exists to prove it works. The FedRAMP PMO, the 3PAO (CyberAudit Partners LLC), and the Authorizing Official all rely on the SSP as the authoritative source of truth about the system's security posture.

In a real engagement, producing an SSP of this quality costs between $150,000 and $300,000 and takes 6–12 months of sustained effort from a senior GRC team.
